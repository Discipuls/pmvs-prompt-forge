# PromptForge

**PromptForge** — Android-приложение для улучшения промптов к нейросетям.

## Краткое описание задачи

Пользователь часто не умеет составлять хорошие запросы к нейросетям: промпт
получается размытым, без указания формата, тона и деталей. PromptForge решает
эту проблему — пользователь вводит черновой запрос, приложение задаёт несколько
уточняющих вопросов (ответы выбираются одним кликом) и через внешний LLM-API
генерирует готовый улучшенный промпт. История промптов хранится локально, а
фидбек пользователя постепенно подстраивает стиль генерации под него.

Приложение реализовано на Kotlin + Jetpack Compose, данные хранятся в SQLite
через Room, сетевые запросы выполняются асинхронно через корутины.
