# PromptForge

## Project Name

**PromptForge** — Android-приложение для улучшения промптов к нейросетям.

Учебный проект по дисциплине ПМВС (БГУ, ФПМИ), команда `pmvs11a-project-diva`.

## Description

PromptForge помогает превратить черновую идею в качественный промпт для нейросети.
Пользователь вводит короткий черновой запрос, приложение через внешний LLM-API
задаёт 2–3 уточняющих вопроса (ответы выбираются в один клик), а затем генерирует
финальный улучшенный промпт. Все промпты сохраняются в локальную базу данных, их
можно просматривать, оценивать и удалять. Приложение запоминает фидбек
пользователя и подстраивает стиль генерации под его предпочтения.

Основные возможности:

- ввод чернового промпта и пошаговое уточнение через вопросы-кнопки;
- генерация улучшенного промпта и копирование результата в буфер обмена;
- история всех промптов с оценкой (1–5) и датой;
- обучение на фидбеке — комментарии пользователя меняют инструкции для ИИ;
- локальное хранение данных без аккаунтов и регистрации.

## Installation

Как установить и запустить приложение локально:

1. Установите [Android Studio](https://developer.android.com/studio)
   (версия Ladybug или новее) и JDK 11.
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/fpmi-pmvs2026/pmvs11a-project-diva.git
   ```
3. Откройте папку проекта в Android Studio (`File → Open`) и дождитесь
   синхронизации Gradle.
4. Получите бесплатный API-ключ на [OpenRouter](https://openrouter.ai/keys).
5. Укажите ключ в файле
   `app/src/main/java/com/example/promptforge/api/GeminiApi.kt` —
   замените значение переменной `key`:
   ```kotlin
   private val key = "ВАШ_OPENROUTER_API_KEY"
   ```
6. Подключите Android-устройство (с включённой отладкой по USB) или запустите
   эмулятор (минимальная версия Android — 7.0, API 24).
7. Нажмите **Run ▶** в Android Studio.

Сборка APK из командной строки:

```bash
./gradlew assembleDebug
```

Готовый файл появится в `app/build/outputs/apk/debug/`.

## Usage

После установки:

1. На главном экране отображается история промптов. При первом запуске она пустая.
2. Нажмите кнопку **+** (FAB), чтобы создать новый промпт.
3. Введите черновой запрос (например, «напиши письмо про встречу») и нажмите
   **Improve**.
4. Ответьте на 2–3 уточняющих вопроса — просто нажимайте на нужный вариант.
5. Получите улучшенный промпт. Его можно скопировать кнопкой **Copy**.
6. Поставьте оценку (1–5 звёзд) и при желании оставьте комментарий —
   он поможет приложению лучше подстроиться под ваш стиль. Нажмите **Save**.
7. Любой сохранённый промпт можно открыть из истории, чтобы посмотреть детали,
   или удалить (иконка корзины).
